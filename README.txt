Smart Random Music Player (SRMP)
Developed by: github.com/nielsx

=== Español:

- Permite reproducir canciones de forma aleatoria evitando al máximo la extraña repetición de algunas canciones que uno piensa ya haber escuchado antes, un día anterior o hace unas horas, y teniendo miles de canciones eso no debería pasar.

- Sólo las canciones que escuchamos más de la mitad de la canción serán añadidas a una base de datos local, sincronizada cuando queramos a una base de datos en la nube. No se añaden todas las canciones porque a veces demoramos casi un minuto en pasarla porque no nos gusta o queremos algo diferente en ese momento.

- Las listas de reproducción serán creadas automáticamente simplemente agrupando temporalemente las canciones de la base de datos. Nuestro humor será el mejor realizador automático de listas.

- El programa no permite la reproducción contínua, es decir que cuando terminar una canción es necesario que el usario de un nuevo clic para escuchar otra canción aleatoria, esto como solución temporal para evitar acceso a la base de datos de música favorita de canciones que sin darnos cuenta y sin que nos gusten mucho las estemos ecuchando.

- Más adelante el programa permitirá vincular la música que escuchamos en el celular, en YouTube, SoundCloud, Spotify, etc. para almacenar una sola grande base de datos personal.

- La base de datos tendrá una lista de las canciones reproducidas usando SRMP, y como datos asociados a cada una estará el número de veces que ha sido escuchada, el tiempo si decidimos no terminar de esucharla y optamos por esuchar otra, la hora del día en que escuchamos ese tipo de canciones, etc.

- Finalmente el programa permitirá sugerir canciones aleatorias de dos tipos: las favoritas que obviamente serán aquellas que más hemos escuchado, y las que casi nunca escuchamos, creando así listas de reproducción totalmente acordes a lo que queremos escuchar y sin complicaciones.
